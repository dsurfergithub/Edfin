# Edfin

Simulador interactivo de educación financiera y decisiones de vida inspirado en la parábola de los talentos y en la psicología del dinero.

## Qué incluye esta primera versión
- Pantalla inicial con nombre y edad
- Decisiones por desplegable
- Variables básicas: dinero, conocimiento, libertad, salud y disciplina
- Línea temporal de decisiones
- Cierre con un Índice de Talentos

## Cómo ejecutarlo
Abre `index.html` directamente en el navegador o publícalo con GitHub Pages.
